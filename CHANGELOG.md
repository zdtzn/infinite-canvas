+ [新增] 增加 HTTPS/Caddy 部署方案、容器健康检查、安全响应头、限流、请求 ID 和结构化日志。

+ [修复] 本地 Agent 请求与发起标签页强绑定，校验工具结果归属，并在页面关闭后回退到最近聚焦页面。
